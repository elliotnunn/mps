Welcome to HTMLSample

HTMLSample is an application illustrating how to use the new HTMLRenderingLib services found in Mac OS 9.0. HTMLRenderingLib is Apple's light-weight HTML rendering engine capable of displaying HTML 4.3 files.

Resources included in this package are as follows:

  • A HTML version of the documentation for the HTMLRenderingLib,
  • Online help for this application,
  • A guide for extending the capabilities off this application.
